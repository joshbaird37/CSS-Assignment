CSS Topic - Font

There are several different ways to alter the font to change how it appears to the user. Three of the most common ways are by making text bold, italics or changing font size. For each of these there are several ways to make the text that way.

How to make text Bold

There are several different ways to change a font to bold. The two key words are bold and strong. Either of these words will change the font to become bold. Also, you can add these key words right before the word you want bolded in the html or you can have it apply to the entire type by adding it to the CSS.

See the Pen Untitled by Josh (@jbaird37) on CodePen.

How to make text Italics

There are also several different ways to change a font to italics. The two key words are i (for italics) and em (for emphasis). Either of these words will change the font to become italics. Just like bold, you can add these key words right before the word you want italics in the html or you can have it apply to the entire type by adding it to the CSS.

See the Pen Italics by Josh (@jbaird37) on CodePen.

How to change font size

Lastly font size. Just like the other two there are many ways to change the size of the font. The best way is assigning it with the css using one of 3 methods. When assigning font-size you can either assign px pixels, em a multiplier of pixels, or vw viewport width which changes with different desktops.
